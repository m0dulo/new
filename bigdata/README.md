## 程序使用方法

* 安装 [Node.js](https://nodejs.org/)
* `cd` 进入本文件所在的目录，运行
```bash
$ nodejs
> require('./analysis.js')
```
* 程序会输出10条数据用于检查。一个名为 `a` 的数组会被放到 `GLOBAL` 中。如果一切正常的话，可以开始分析了。
```javascript
> a.length
13830
> // 统计各个微信公众号发送的弹幕数
> a.reduce(function (r, e) { r[e.official_account]++; return r; }, [0, 0])
[ 12661, 1169 ]
```
